# 💰 Money Practice Webapp

A fun, colorful webapp for 3rd graders to practice adding and subtracting money!

## 🎮 Features

### Game Modes
- **➕ Addition** - Practice adding money amounts
- **➖ Subtraction** - Practice subtracting money amounts
- **🔀 Mixed** - Random mix of both operations

### Difficulty Levels
| Level | Max Amount | Perfect For |
|-------|------------|-------------|
| Easy | Up to $5 | Beginners |
| Medium | Up to $20 | Grade level |
| Hard | Up to $50 | Advanced practice |

### Session Options
Choose how many questions per session:
- 10, 20, 50, 100, or 200 questions

### Engagement Features
- 🏆 Score tracking with streak bonuses
- 🪙 Visual money display (pennies, nickels, dimes, quarters, bills)
- 📊 Real-time progress tracking
- 🎉 End-of-session summary with accuracy stats
- 📱 Mobile responsive design

## 📁 Usage

### Option 1: Open Directly
Simply open `index.html` in any web browser.

### Option 2: Local Server
```bash
python -m http.server 8080
```
Then visit `http://localhost:8080`

### Option 3: GitHub Pages
Access the hosted version at the link above.
